1. Project Introduction

This project is built using Python3.9, with Django and MySQL.

When using the YOLOV5 network, you need to install some data analysis libraries.

2. Installation Steps

First, create a new directory in Terminal:

mkdir newfile

Then enter the directory:

cd newfile

Next, clone the project into the current directory:

git clone git@github.com:isKage/mis.git

Go into the project root directory:

cd mis

3. Install Dependencies

Note: This project relies on Python3.9

pip install -r requirements.txt

4. MySQL Configuration

Create a local_settings.py file in the mis directory with the following settings:

Set Language

LANGUAGE_CODE = 'zh-hans'

MySQL Configuration

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'mis',  # Use your own database name
        'USER': 'root',  # Database user
        'PASSWORD': '******',  # Database password
        'HOST': '127.0.0.1',  # Localhost
        'PORT': '3306',  # Default port
    }
}

Next, run the following commands to create the database tables:

python3 manage.py makemigrations
python3 manage.py migrate

5. Run the Project

Run the following command in Terminal:

python3 manage.py runserver

Then, open the following link in your browser, if running locally:

http://localhost:8000/
