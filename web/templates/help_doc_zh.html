1. 项目简介

该项目基于Python3.9，使用Django和Mysql构建。

在使用YOLOV5网络时，您需要安装一些数据分析相关的库。

2. 安装步骤

首先，在Terminal中创建一个新文件夹：

mkdir newfile

然后进入该文件夹：

cd newfile

接下来，克隆该项目到当前目录：

git clone git@github.com:isKage/mis.git

进入项目根目录：

cd mis

3. 安装依赖

注意：该项目依赖于Python3.9

pip install -r requirements.txt

4. Mysql 设置

在 mis 目录下创建一个local_settings.py文件，并进行如下设置：

设置语言

LANGUAGE_CODE = 'zh-hans'

Mysql 设置

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'mis',  # 使用你自己的数据库名称
        'USER': 'root',  # 数据库用户
        'PASSWORD': '******',  # 数据库密码
        'HOST': '127.0.0.1',  # 本地数据库
        'PORT': '3306',  # 默认端口
    }
}

接下来，运行以下命令创建数据表：

python3 manage.py makemigrations
python3 manage.py migrate

5. 启动项目

在Terminal中运行以下命令：

python3 manage.py runserver

然后，在浏览器中打开如下链接，若在本地运行：

http://localhost:8000/
